# v3.0 更新说明

## 新功能

- **用户仪表盘** — 全新左右结构界面（左侧功能菜单 + 右侧内容区），替代原有单一首页
- **欢迎页** — 首页默认展示欢迎语、登录状态和功能引导卡片
- **功能菜单导航** — 左侧固定侧边栏，包含长链接转短链接、短链接转长链接、访问统计、批量生成、二维码生成等菜单项
- **权限分级** — 未登录用户可使用"长链接转短链接"基础功能，其他菜单置灰并提示登录；登录用户可访问全部功能
- **自定义后缀可用性优化** — 输入框始终可见，未登录时输入自定义后缀实时提示"登录后可使用自定义后缀"
- **登录后跳转优化** — 普通用户登录后跳转首页仪表盘，管理员仍跳转管理后台

## 界面优化

- **深色玻璃侧边栏** — 深灰蓝底色、琥珀色高亮选中态、毛玻璃质感
- **登录状态条** — 内容区顶部固定显示圆点状态标识 + 当前登录状态文字
- **动效** — 菜单切换淡入上移动画、Toast 从顶部滑入提示、侧边栏选中态琥珀色指示条
- **统一色彩系统** — 新增 `--sidebar-*`、`--accent-*`、`--text-*`、`--bg-main` 等 CSS 变量
- **品牌字体** — Logo 使用 Plus Jakarta Sans 字体，短链码使用 JetBrains Mono 等宽字体
- **占位页面** — 统一"功能开发中"占位卡片，展示功能预告

## 重构

- **路由重构** — 根路径 `/` 改为 UserLayout 父级路由，包含 6 个子路由（欢迎页 + 5 个功能页）
- **AppHeader 简化** — 移除首页和管理后台导航链接（侧边栏已承载导航）
- **删除 Home.vue** — 内容迁移至 `LongToShort.vue`，消除冗余

# v2.0 更新说明

## Bug 修复

- **概览页统计为 0** — 匿名创建的短链接未计入统计，查询条件增加 `OR userId IS NULL`
- **今日访问量始终为 0** — 硬编码占位值未替换，改为从 `daily_stats` 表实时查询当日 PV
- **硬删除导致统计丢失** — 删除短链接改为软删除（`is_active = false`），保留历史访问数据
- **概览总短链数不准确** — 改为只统计活跃（`is_active = true`）的短链接
- **数据统计图表空白** — 修复两处 bug：
  - 前端时序错误：`loading` 置 false 后未等 DOM 渲染就初始化图表
  - API 字段名不匹配：前端读取 `d.date`，实际返回字段为 `d.statsDate`
- **短码生成冲突过多** — Bloom Filter 重试耗尽后直接抛异常，改为返回末次尝试结果，由数据库唯一索引兜底
- **切换天数后图表空白** — 切换 7/14/30 天时 `loading` 切换导致图表容器 DOM 重建，旧 ECharts 实例指向已分离元素；修复为每次渲染前释放旧实例
- **UV / IP 始终为 0** — `daily_stats` 的 upsert 语句只更新了 PV，新增新访客判断逻辑并补充 UV 和 IP 自增更新

## 优化

- **自定义后缀实时校验** — 输入框即时提示格式错误（字母开头、4-16 位字母数字）
- **Vercel 设计主题** — 全局颜色、间距、圆角、阴影全面替换为 Vercel 风格，重构 header / sidebar / 首页 / 管理后台样式
- **图表指标说明** — 统计页下方增加 PV / UV / IP 含义说明

## Docker

- **清理无用环境变量** — docker-compose 中后端服务移除未使用的 `DB_HOST`/`DB_PORT`/`DB_USER`/`DB_NAME`/`REDIS_HOST`/`REDIS_PORT`，只保留实际生效的配置
- **启动顺序优化** — 前端 `depends_on` 改为标准语法

## 安全

- **移除密码硬编码** — MySQL 密码改为 `${MYSQL_PASSWORD:root123}` 环境变量占位符，真实密码移入 `~/.bashrc` 本地管理
- **清理 Git 追踪** — `.claude/` 加入 `.gitignore`，移除包含本地路径的配置文件

# v2.0 更新说明

## Bug 修复

- **概览页统计为 0** — 匿名创建的短链接未计入统计，查询条件增加 `OR userId IS NULL`
- **今日访问量始终为 0** — 硬编码占位值未替换，改为从 `daily_stats` 表实时查询当日 PV
- **硬删除导致统计丢失** — 删除短链接改为软删除（`is_active = false`），保留历史访问数据
- **概览总短链数不准确** — 改为只统计活跃（`is_active = true`）的短链接
- **数据统计图表空白** — 修复两处 bug：
  - 前端时序错误：`loading` 置 false 后未等 DOM 渲染就初始化图表
  - API 字段名不匹配：前端读取 `d.date`，实际返回字段为 `d.statsDate`
- **短码生成冲突过多** — Bloom Filter 重试耗尽后直接抛异常，改为返回末次尝试结果，由数据库唯一索引兜底
- **切换天数后图表空白** — 切换 7/14/30 天时 `loading` 切换导致图表容器 DOM 重建，旧 ECharts 实例指向已分离元素；修复为每次渲染前释放旧实例
- **UV / IP 始终为 0** — `daily_stats` 的 upsert 语句只更新了 PV，新增新访客判断逻辑并补充 UV 和 IP 自增更新

## 优化

- **自定义后缀实时校验** — 输入框即时提示格式错误（字母开头、4-16 位字母数字）
- **Vercel 设计主题** — 全局颜色、间距、圆角、阴影全面替换为 Vercel 风格，重构 header / sidebar / 首页 / 管理后台样式
- **图表指标说明** — 统计页下方增加 PV / UV / IP 含义说明
